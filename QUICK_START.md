# 快速开始指南 🚀

欢迎使用拼豆豆管理系统！这里是最快速的上手指南。

## 🎯 你想用哪个版本？

### 选项 1：网页版（推荐）⭐

**优点**：
- ✅ 打开浏览器就能用
- ✅ 多设备自动同步
- ✅ 数据安全存储在云端
- ✅ 随时随地访问

**需要**：
- 注册一个账号（免费）
- 有网络连接

**立即使用**：
1. 打开 https://loismeng-qh.github.io/pindou_manager/
2. 点击"注册"创建账号
3. 登录开始使用

**详细配置**：参考 `WEB_CLOUD_GUIDE.md`

---

### 选项 2：微信小程序

**优点**：
- ✅ 在微信中使用
- ✅ 可以拍照上传图纸
- ✅ 数据云端同步
- ✅ 手机端体验更好

**需要**：
- 配置服务器 IP 地址
- 注册微信小程序（可选，用测试号也行）
- 下载微信开发者工具

**配置步骤**：
1. 获取服务器 IP
2. 修改 `miniprogram/app.js` 中的 API 地址
3. 修改 `miniprogram/project.config.json` 中的 AppID
4. 用微信开发者工具导入
5. 点击"编译"运行

**详细配置**：参考 `MINIPROGRAM_SETUP.md`

---

### 选项 3：单文件版（离线）

**优点**：
- ✅ 不需要网络
- ✅ 不需要登录
- ✅ 一个文件全包含
- ✅ 可以保存到本地

**缺点**：
- ❌ 数据只在本地
- ❌ 不能跨设备同步

**立即使用**：
打开 https://loismeng-qh.github.io/pindou_manager/pindou-manager-standalone.html

或下载到本地：
```bash
wget https://loismeng-qh.github.io/pindou_manager/pindou-manager-standalone.html
# 然后用浏览器打开
```

## 🎬 网页版快速开始（3分钟）

### 第一步：打开网页

访问：https://loismeng-qh.github.io/pindou_manager/

### 第二步：注册账号

1. 点击页面下方的"注册新账号"
2. 输入信息：
   - **邮箱**：test@example.com（你的邮箱）
   - **密码**：123456（至少6位）
   - **用户名**：小明（你的昵称）
3. 点击"注册"按钮

### 第三步：登录

注册成功后会自动登录，或手动输入邮箱密码登录。

### 第四步：添加第一个豆子

1. 点击顶部"豆子管理"标签
2. 点击右上角"添加豆子"按钮
3. 填写信息：
   - **色号**：A1（或点"选择"从预设中选）
   - **颜色名称**：淡黄
   - **数量**：100
4. 点击"保存"

### 第五步：添加第一个图纸

1. 点击顶部"图纸管理"标签
2. 点击右上角"添加图纸"按钮
3. 填写信息：
   - **图纸名称**：樱花图案
   - **描述**：粉色樱花（可选）
   - **选择图片**：从电脑选一张图片
4. 点击"保存"

### 第六步：查看设置

1. 点击顶部"设置"标签
2. 可以切换主题颜色
3. 查看关于信息

### 完成！🎉

现在你可以：
- 添加更多豆子
- 调整豆子数量（+1, +10, -1, -10）
- 添加更多图纸
- 切换想拼/已拼状态
- 点击🔄同步数据
- 点击⬅退出登录

## 📱 小程序快速开始（15分钟）

### 前置要求

- 有一台服务器（腾讯云轻量应用服务器）
- 服务器上已经部署了后端 API
- 获取了服务器的公网 IP 地址

### 配置步骤

#### 1. 修改 API 地址（1分钟）

编辑文件 `miniprogram/app.js`，第 2 行：

```javascript
const API_BASE_URL = 'http://123.456.789.0:3000';
```

替换 `123.456.789.0` 为你的服务器 IP。

#### 2. 获取 AppID（5分钟）

1. 访问 https://mp.weixin.qq.com/
2. 注册小程序账号（个人免费）
3. 登录后，在 **开发 > 开发设置** 找到 AppID
4. 复制 AppID

#### 3. 修改配置文件（1分钟）

编辑文件 `miniprogram/project.config.json`，第 45 行：

```json
"appid": "wx1234567890abcdef"
```

替换为你的 AppID。

#### 4. 下载开发工具（5分钟）

1. 访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载并安装微信开发者工具
3. 打开工具，选择"小程序"
4. 点击"导入项目"
5. 选择 `miniprogram` 文件夹
6. 填入 AppID
7. 点击"导入"

#### 5. 开发调试（2分钟）

1. 在工具中点击"详情"
2. 勾选"不校验合法域名"
3. 点击"编译"
4. 在左侧模拟器看到小程序界面

#### 6. 测试功能（1分钟）

1. 注册账号
2. 添加豆子
3. 上传图纸
4. 查看设置

### 完成！🎉

更多详细信息，参考 `MINIPROGRAM_SETUP.md`

## 🆘 遇到问题？

### 网页版问题

**Q: 登录失败怎么办？**
A: 
1. 检查网络连接
2. 确认邮箱和密码正确
3. 尝试重新注册

**Q: 数据看不到？**
A: 
1. 点击右上角🔄刷新按钮
2. 检查网络连接
3. 退出重新登录

### 小程序问题

**Q: API 连接失败？**
A:
1. 检查 API 地址是否正确
2. 确认服务器正在运行
3. 检查防火墙端口 3000 是否开放
4. 确认勾选了"不校验合法域名"

**Q: 无法注册账号？**
A:
1. 检查邮箱格式
2. 密码至少6位
3. 查看控制台错误信息

### 单文件版问题

**Q: 打开是空白？**
A:
1. 确认浏览器支持（Chrome, Firefox, Edge）
2. 清除浏览器缓存
3. 尝试重新下载文件
