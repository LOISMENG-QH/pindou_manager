```
/root/.openclaw/workspace/pindou-manager/pindou-manager-standalone.html
```

## 🚀 使用方法

### 方法1：直接下载使用（推荐）

1. **下载文件**
   ```bash
   # 在你的本地电脑上下载这个文件
   scp root@你的服务器IP:/root/.openclaw/workspace/pindou-manager/pindou-manager-standalone.html ~/Downloads/
   ```

2. **双击打开**
   - 直接双击 `pindou-manager-standalone.html` 文件
   - 会在浏览器中打开（推荐使用 Chrome、Edge、Firefox）
   - 所有功能完全可用！

3. **添加到收藏/桌面**
   - 可以添加书签方便下次使用
   - 也可以创建桌面快捷方式

### 方法2：放到任何 Web 服务器

把这个文件上传到任何网站空间都能使用：
- GitHub Pages
- Netlify

- 你自己的服务器
- 甚至网盘的在线预览

### 方法3：分享给朋友

## ✨ 功能说明

### 豆子管理
- ✅ 添加色号（色号、名称、数量、提醒阈值）
- ✅ 快速调整数量（+/- 按钮）
- ✅ 编辑和删除色号
- ✅ 低库存自动提醒

### 图纸管理
- ✅ 上传拼豆图纸图片
- ✅ 自动生成缩略图
- ✅ 全屏查看图片
- ✅ 添加图纸描述和备注

### 数据存储
- ✅ 使用 IndexedDB 浏览器数据库
- ✅ 数据永久保存（除非清除浏览器数据）
- ✅ 完全离线可用
- ✅ 图片完整保存在本地

## 📱 设备支持

- ✅ 电脑（Windows / macOS / Linux）
- ✅ 手机（iOS / Android）
- ✅ 平板电脑
- ✅ 需要支持 IndexedDB 的现代浏览器

## 💡 使用技巧

1. **手机使用**
   - 打开网页后，点击浏览器菜单
   - 选择"添加到主屏幕"
   - 就像一个真正的 App 一样使用！

2. **数据备份**
   - 数据存在浏览器本地
   - 清除浏览器数据会导致丢失
   - 建议定期导出重要图纸（截图保存）

3. **多设备使用**
   - 可以在多个设备上使用同一个文件
   - 但每个设备的数据是独立的
   - 需要手动同步
